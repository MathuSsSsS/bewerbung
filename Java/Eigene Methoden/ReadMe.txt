Expended Form: 
Eine Methode, die ein Int als Parameter übergeben bekommt und dieses Int dann in der erweiterten Form ausgibt. Z.B. wird 5012 zu "5000 + 10 + 2" und 399 wird zu "300 + 90 + 9"

RegExCheck:
Methoden, welche überprüfen können, ob eine Zeichenkette vordeffinierten Regeln entspricht. So kann man z.B. überprüfen, ob eine Email-Adresse korrekt angegeben ist oder eine Telefonnummer ein gültiges Format hat. 
Bei der Datumsprüfung habe ich nicht nur drauf geachtet, dass das Datum das richtige Format hat. Monate haben eine unterschiedliche Anzahl von Tagen, außerdem müssen auch Schaltjahre in Betracht gezogen werden.
