Waldspiel: 
Kleines Konsolenbasiertes Spiel, bei dem ich die Objektorientation geübt habe. Der Spieler hat eine Begegnung und muss sein nächstes Handeln auswählen, ohne vorher zu wissen, wer oder was ihm begegnet. Je nach Entscheidung kann der Spieler einen Lebenspunkt verlieren. Das geht solange weiter, bis die Lebenspunkte des Spielers auf 0 sind.

Weltrangliste:
Praktische Anwendung des Bubble Sorts. In diesem Beispiel habe ich eine vorgegebene Liste an Athleten. Die Objekte der Klasse werden bei Ausgabe in einen speziellen String umgewandelt. Die Liste wird erst durchgemischt und ausgegeben, und dann sortiert und wieder ausgegeben.
